## RECOMMENDED DESIGN FOR NEXT SESSION

1. **Icon set:** Lucide (lucide.dev) — open source, 1000+ icons, consistent 24×24 viewBox, widely used in React/Next.js apps. Use inline `<svg>` (not font) so they respect `currentColor`.
2. **Sizing:**
   - Inline with text: `width:1em; height:1em; vertical-align:-0.1em`
   - Decorative standalone: 48-64px
   - Toast/error indicators: 20px
3. **Color:**
   - Monochrome white (#ffffff) for most
   - Red accent (#cc3333 — existing brand) for primary/success CTAs
   - Muted gray (rgba(255,255,255,0.7)) for secondary
4. **Positioning rule (from user):** ALWAYS at END of text, never start. Example:
   - ❌ OLD: `📝 הפרטים שלך:`
   - ✅ NEW: `הפרטים שלך: <svg>...</svg>`
5. **Suggested Lucide mappings:**
   - ✅ success → `<check-circle>`
   - ❌ error → `<x-circle>`
   - ⏳ loading → `<loader>` (with spin animation)
   - 🔑 login → `<key>`
   - 📝 details → `<file-text>`
   - 🎬 video → `<play-circle>`
   - 💬 chat → `<message-circle>` (keep for WhatsApp button — brand color #25D366)
   - 🎓 education → `<graduation-cap>`
   - 🚀 launch → `<rocket>`
   - ⚡ latest → `<zap>`
   - 🔥 streak → `<flame>`
   - 😊 greeting → REMOVE (social cheese, not needed)
   - 🌟 new → `<sparkles>` OR just text badge
